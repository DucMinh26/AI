**QUY TRÌNH CÁC BƯỚC TẠO RA  AI**

* **B1: Build (Xây dựng):** Dùng `Sequential()`, `add(Dense...)` -> Thiết kế cái khung.
* **B2: Bind (Kết nối/Biên dịch):** Dùng `.compile()` -> Cài đặt luật chơi và cách chấm điểm.
* **B3l Burn (Đốt cháy/Huấn luyện):** Dùng `.fit(X_train, y_train)` -> Đây mới là lúc dữ liệu đổ vào, các nơ-ron bắt đầu "nóng" lên để học.
